# eclpy documentation

`eclpy` evaluates Common Lisp from Python by hosting ECL inside a packaged
WebAssembly runtime. These pages are both the user guide and the source-oriented
reference for maintainers who need to reason about the Python, C, and Lisp
boundaries together.
